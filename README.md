# Workinabox

This repo holds organization-wide documentation and defaults for the sibling repositories in the Workinabox workspace.

## Repos

- `.github` — shared docs, health files, and org-level agent instructions
- `dev` — Rust CLI/TUI for org monitoring and release orchestration
- `ui` — Rust/Leptos web and desktop frontend
- `backend` — Rust backend service with signaling, SFU, and optional local transcription
- `app` — React Native mobile client
